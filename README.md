## Setup

To run this portfolio app locally, follow these steps:

1. Clone the repository: `git clone [https://github.com/L3VU5/portfolio-paweljedrasik]`
2. Navigate to the project folder: `cd portfolio-paweljedrasik`
3. Install dependencies: `npm install`
4. Start the application: `npm run start`
